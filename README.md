## Overview

**Edumed** is a learning platform that provides paid courses for dentistry and other university programs. The platform allows users to discover, subscribe to, and access courses securely, while giving administrators and support staff full control over course management.

The application ensures content security and user privacy through advanced restrictions, making it a professional solution for educational institutions and learners.

---

## Features

### For Users:
- **Course Access:** View all subscribed courses in the "My Courses" section.  
- **Discover Courses:** Browse all available courses in the "Discover" section.  
- **Profile Management:**  
  - Update profile details (name, password).  
  - Change app theme and language.  
  - Contact support directly from the app.  
  - Delete account if desired.  
- **Security:**  
  - Screen capture and recording protection.  
  - Single device login restriction to prevent account sharing.

### For Admin & Support:
- **Course Management:**  
  - Add new courses.  
  - Edit, delete, or update existing courses.  
  - Activate or assign courses to specific users.  
- **User Support:** Assist users with course subscriptions and payments.

---

## Architecture & Technology
- **Frontend:** Flutter  
- **Backend:** Firebase Authentication, Firestore (for courses and user data)  
- **State Management:** Cubit/BLoC (for responsive UI & data handling)  
- **Security Features:**  
  - Protection against screen recording and screenshots.  
  - Single-session login enforcement.

---

## How It Works
1. Users contact support to subscribe to paid courses.  
2. Support activates the purchased courses, which then appear in the user's "My Courses" section.  
3. Users can browse available courses in "Discover," manage their profile, and interact securely with the app.  
4. Admins and support staff have full control to manage courses and users efficiently.
